# LunarDiscord (BETA)

---
> This plugin is currently in beta, so please be patient!
## What is this?
LunarDiscord is a Minecraft to Discord, Minecraft plugin for paper servers running 1.20.4+ 



## Features
Lunar Discord posses multiple features which include:
- /link command
- /unlink (admin/perm "lunardiscord.unlink")
- Discord to Minecraft chat
- Minecraft console channel
- linking channel + modals
- SQLite and Skript possible storage providers
- Skript capabilities
  - Needs the Minecraft plugin Skript
  - variables are customizeable in config.yml
- Linking rewards that are fully customizable

## Requirements:
For this plugin to work you will need Luckperms, and if you enabled Skript storage provider you'll need Skript aswell.
